# Plan: MD Helper – statická webová aplikace

## Cíl
Vytvořit jeden soubor `md-helper.html` – markdown editor s live preview, toolbarem, akcemi a cheat sheetem. Vše v češtině, dark-only design, bez backendu.

---

## Soubor
`md-helper.html` – vše v jednom souboru

---

## Závislosti (CDN)
- **Tailwind CSS** – `https://cdn.tailwindcss.com`
- **Marked** – `https://cdn.jsdelivr.net/npm/marked/marked.min.js`
- **DOMPurify** – `https://cdn.jsdelivr.net/npm/dompurify/dist/purify.min.js`

---

## Struktura HTML

```
<html>
  <head> – meta, title, CDN skripty, custom CSS (prose styles, scrollbar, etc.)
  <body class="bg-gray-950 text-gray-100">
    <header> – "MD Helper" + popis
    <main class="grid grid-cols-2 gap-4"> – 2 sloupce
      <section> – Editor (toolbar + textarea + akce)
      <section> – Live preview
    </main>
    <section> – Cheat sheet

```

## Layout detaily

### Header
- Název "MD Helper" (velký, bílý)
- Podtitulek: "Jednoduchý editor Markdownu s živým náhledem"

### Hlavní obsah (desktop: 2 sloupce, mobil: stacked)
- `lg:grid-cols-2` – na desktopu vedle sebe, na mobilu pod sebou
- Stejná výška obou sloupců: `h-[calc(100vh-...)]` nebo pevná výška

### Editor (levý sloupec)
1. **Toolbar** – řada tlačítek: H1, H2, Bold, Italic, Bullet, Numbered, Checklist, Link, Code, Code Block, Table
2. **Textarea** – monospacovaný font, tmavé pozadí, celá dostupná výška
3. **Řádek akcí** – input pro název souboru + tlačítka: Copy Markdown, Download .md, Reset

### Live preview (pravý sloupec)
- Renderovaný HTML z Marked + DOMPurify
- Stylizované headings, listy, code bloky, tabulky (vlastní CSS prose styly)

## Toolbar akce (JavaScript)

Každé tlačítko volá `insertMarkdown(before, after, placeholder)`:
- zjistí `selectionStart` / `selectionEnd` textarea
- pokud je text označen → obalí ho
- pokud ne → vloží `before + placeholder + after`

| Tlačítko      | Vkládá                                              |
|---------------|-----------------------------------------------------|
| H1            | `# ` na začátek řádku                               |
| H2            | `## ` na začátek řádku                              |
| Bold          | `**text**`                                          |
| Italic        | `_text_`                                            |
| Bullet list   | `- položka\n`                                       |
| Numbered list | `1. položka\n`                                      |
| Checklist     | `- [ ] První úkol\n- [x] Hotovo\n`                 |
| Link          | `[text](url)`                                       |
| Inline code   | `` `kód` ``                                         |
| Code block    | ` ```\nkód\n``` `                                   |
| Table         | Jednoduchá 2×2 tabulka s hlavičkou                  |

---

## Akce (tlačítka pod editorem)

- **Copy Markdown** – `navigator.clipboard.writeText(textarea.value)`, tlačítko dočasně zobrazí "Zkopírováno ✓"
- **Download .md** – vytvoří `Blob`, dynamický `<a>` s `download="${filename}.md"`, klikne, odstraní
- **Reset** – `confirm("Opravdu chceš smazat obsah?")`, pokud ano → vymaže textarea, obnoví výchozí obsah

## Default obsah textarea (ukázkový text)
Při načtení bude v textarea předvyplněný krátký markdown demonstrující různé prvky (heading, tučný, list, odkaz, kód).

## Live preview

```js
function updatePreview() {
  const raw = textarea.value;
  const html = DOMPurify.sanitize(marked.parse(raw));
  previewEl.innerHTML = html;
}
textarea.addEventListener('input', updatePreview);
```

Volat také při inicializaci.

## Cheat sheet (pod hlavním obsahem)

Sekce s kartičkami nebo tabulkou – 8 prvků:

| Prvek       | Syntax                          | Kdy použít            | Příklad (render)         |
|-------------|----------------------------------|-----------------------|--------------------------|
| Nadpis H1   | `# Nadpis`                       | Hlavní nadpis stránky | velký nadpis             |
| Nadpis H2   | `## Podnadpis`                   | Sekce dokumentu       | menší nadpis             |
| Tučný text  | `**text**`                       | Zdůraznění            | **text**                 |
| Kurzíva     | `_text_`                         | Jemné zdůraznění      | _text_                   |
| Seznam      | `- položka`                      | Výčty                 | bullet list              |
| Checklist   | `- [ ] úkol`                     | To-do listy           | checkbox                 |
| Odkaz       | `[text](url)`                    | Hypertextový odkaz    | klikatelný text          |
| Kód         | `` `kód` `` / ` ```blok``` `     | Příkazy, kód          | monospace blok           |
| Tabulka     | `\| sloupec \| ...`              | Přehledná data        | tabulka                  |

Každá karta: syntax, popis, příklad – vizuálně odděleno, tmavé karty.

## CSS / Design

Tailwind dark třídy + custom `<style>` blok pro:
- **prose** styly preview (h1–h6, ul, ol, code, pre, table, blockquote)
- **scrollbar** styling
- **toolbar tlačítka** – hover, active stavy
- **textarea** – monospace font, resize-y, no outline ring
- **tabulka v preview** – border, zebra stripes nebo kontrast

Barvy (Tailwind):
- Pozadí: `gray-950` / `gray-900`
- Karty/panely: `gray-800` / `gray-850`
- Bordery: `gray-700`
- Text: `gray-100` / `gray-400`
- Akcenty: `indigo-500` / `indigo-400`
